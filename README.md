# 🌱 Agrox Coin (AGX)

**A criptomoeda do agronegócio brasileiro. Nascida no Rio Grande do Sul para o Brasil e o mundo.**

## Características
- **Algoritmo:** RandomX (CPU mining)
- **Tempo de bloco:** 10 minutos
- **Recompensa:** 50 AGX por bloco
- **Halving:** A cada 5 anos (262.800 blocos)
- **Supply máximo:** 39.000.000 AGX
- **Privacidade:** RingCT, Stealth Addresses (herdado da Monero)

## 🌎 Por que Agrox?
Criada para ser a moeda digital do agronegócio brasileiro — o setor que representa 25% do PIB nacional. Privacidade para transações comerciais, mineração justa e supply previsível.
